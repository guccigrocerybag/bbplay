# 📜 Все промты для разработки BBplay

> Полный список промтов (запросов к AI), использованных при разработке мобильного приложения BBplay для сети компьютерных клубов BlackBearsPlay.

---

## 1. 🚀 Инициализация проекта

**Промт:**
```
Создай Flutter-приложение для сети компьютерных клубов "BBplay". 
Нужна регистрация, авторизация, бронирование ПК, пополнение баланса, 
чат с поддержкой. Используй Provider для state management.
```

**Результат:** Создан базовый каркас проекта с `main.dart`, `ApiClient`, `UserProvider`, `SettingsProvider`, экранами авторизации, бронирования, профиля, чата.

---

## 2. 🎨 Темизация и локализация

**Промт:**
```
Добавь тёмную и светлую тему, переключение между ними, 
сохранение выбора через shared_preferences. 
Добавь поддержку русского и английского языков через SettingsProvider.
```

**Результат:** Реализована полная темизация (ThemeData для светлой/тёмной темы), переключение языка с сохранением в SharedPreferences, все строки вынесены в `getText('key')`.

---

## 3. 🔐 Авторизация (3-шаговая регистрация)

**Промт:**
```
Реализуй вход по логину/паролю и 3-шаговую регистрацию:
1) POST /api/v2/.../members — создание пользователя
2) POST /request-sms — запрос SMS
3) POST /verify — подтверждение кода
После регистрации — автоматический вход.
```

**Результат:** Создан `AuthScreen` с формой входа/регистрации, SMS-диалогом, валидацией пароля.

---

## 4. 🖥 Бронирование ПК

**Промт:**
```
Создай экран бронирования: выбор клуба из списка (/cafes), 
выбор даты и времени, длительности сессии (30 мин - 5 часов), 
отображение карты ПК с занятостью, расчёт цены.
Используй Row + Expanded для сетки ПК (игнорируя pc_box_left/top).
```

**Результат:** Создан `BookingScreen` с календарём, слайдером времени, сеткой ПК, расчётом цены, `BookingProvider`.
