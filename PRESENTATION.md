# BBplay — Мобильное приложение для сети компьютерных клубов

## 🎯 О проекте

**BBplay** — это кросс-платформенное мобильное приложение (Flutter) для сети компьютерных клубов **BlackBearsPlay**. Приложение позволяет пользователям регистрироваться, пополнять баланс, бронировать игровые ПК, отслеживать свои сессии, участвовать в турнирах, заказывать еду и общаться с поддержкой через AI-чат.

---

## 🏗 Архитектура

| Компонент | Технология |
|---|---|
| **Фреймворк** | Flutter (Dart 3+) |
| **State Management** | Provider (ChangeNotifier) |
| **Навигация** | CurvedNavigationBar + PageController |
| **Сеть** | HTTP (кастомный ApiClient) |
| **Локальное хранение** | SharedPreferences + Hive (кеш) |
| **Анимации** | Кастомные Flutter-анимации (AnimatedBuilder, AnimationController) |
| **AI-чат** | YandexGPT API + Rule-based fallback |
| **Уведомления** | flutter_local_notifications |
| **Оффлайн** | Connectivity Plus + Cache Service |

## ✨ Ключевые возможности

### 🔐 Авторизация и регистрация
- Вход по логину и паролю
- 3-шаговая регистрация с подтверждением по SMS
- Валидация пароля (минимум 8 символов, заглавные/строчные буквы, цифры)
- **SplashScreen** с анимацией логотипа
- **Onboarding** (3 экрана знакомства с приложением)

### 🖥 Бронирование ПК
- Выбор клуба из списка
- Календарь с выбором даты и времени
- Выбор длительности сессии (30 мин — 5 часов)
- Интерактивная карта ПК с отображением занятости
- Умный расчёт цены на основе зоны (Bootcamp, VIP, GameZone)
- **Session Hub** — быстрый доступ к активным сессиям с главного экрана

### 🏆 Турниры
- Список актуальных турниров с датами и призовыми
- Детальная информация о каждом турнире
- Регистрация на турниры

### 🍔 Меню еды
- Категории: Пицца, Закуски, Напитки, Комбо
- Детальная карточка каждого блюда с ценой и описанием
- Удобный интерфейс для заказа

### 📊 История бронирований
- Просмотр всех сессий
- Статусы: UPCOMING, ACTIVE, EXPIRED
- Возможность отмены будущих бронирований

### 👤 Профиль
- Отображение баланса с геймерским рангом (BRONZE/SILVER/GOLD)
- **Цветовая индикация ранга**: золотой (GOLD), серебряный (SILVER), бронзовый (BRONZE)
- Прогресс-бар до следующего ранга
- Аватарка с возможностью смены (камера/галерея)
- Пополнение счёта (с обработкой ошибок сервера)
- Настройки темы (светлая/тёмная)
- Переключение языка (русский/английский)

### 🤖 AI-чат поддержки
- Интеграция с YandexGPT
- Rule-based fallback (работает без API-ключей)
- Контекстные ответы на основе документации клуба
- Быстрые ответы на частые вопросы (цены, адреса, правила)

### 📰 Новости
- Лента новостей клуба
- Парсинг HTML-контента
- Pull-to-refresh

### 🎬 Видеоплеер
- Встроенный плеер для трансляций
- Поддержка YouTube

## 🛡 Надёжность и отказоустойчивость

- **Оффлайн-режим**: кеширование данных через Cache Service
- **Индикатор подключения**: отображение статуса сети (Connectivity Plus)
- **Обработка ошибок сервера**: все API-запросы обёрнуты в try-catch
- **Умная проверка ответов**: сервер может возвращать "success" с кодом 0, 3 или 200; проверяем message на ошибки
- **Локальное состояние**: бронирования сохраняются локально при сбоях сервера
- **Push-уведомления**: напоминания о начале и конце сессии
- **Graceful fallback**: при заблокированных эндпоинтах (/topup, /cancel) показываем понятное сообщение

## 🛠 Технические детали

### API
- **Base URL**: `https://vibe.blackbearsplay.ru`
- **Формат**: JSON (application/json)
- **Аутентификация**: member_name + password → сессионный ключ
- **Подпись запросов**: MD5 (rand_key + key)

### Структура проекта

## 🏆 Почему BBplay достоин победы?

1. **Полноценное production-приложение** с реальной серверной интеграцией
2. **Сложная бизнес-логика**: бронирование, расчёт цен, работа с нестабильным API
3. **Отказоустойчивость**: оффлайн-режим, кеширование, обработка ошибок, fallback UI
4. **AI-интеграция**: чат с YandexGPT для поддержки пользователей
5. **Кастомные анимации**: уникальный UI с анимированным логотипом, спиннером, конфетти
6. **Push-уведомления**: напоминания о сессиях
7. **Кросс-платформенность**: Android, iOS, Web, Windows
8. **Тёмная/светлая тема** и полная локализация (RU/EN)
9. **Турниры и меню еды** — дополнительные функции для вовлечения пользователей
10. **Session Hub** — быстрый доступ к активным сессиям
11. **Чистая архитектура**: Provider, разделение на слои, переиспользуемые виджеты
12. **Работа с нестабильным API**: умная проверка ответов, обработка серверных ошибок
13. **Уникальный брендинг**: собственный логотип, иконки для всех платформ, PWA-манифест
