# BBplay — Мобильное приложение для сети компьютерных клубов

## 🎯 О проекте

**BBplay** — это кросс-платформенное мобильное приложение (Flutter) для сети компьютерных клубов **BlackBearsPlay**. Приложение позволяет пользователям регистрироваться, пополнять баланс, бронировать игровые ПК, отслеживать свои сессии и общаться с поддержкой через AI-чат.

---

## 🏗 Архитектура

| Компонент | Технология |
|---|---|
| **Фреймворк** | Flutter (Dart 3+) |
| **State Management** | Provider (ChangeNotifier) |
| **Навигация** | CurvedNavigationBar + PageController |
| **Сеть** | HTTP (кастомный ApiClient) |
| **Локальное хранение** | SharedPreferences + Hive |
| **Анимации** | Кастомные Flutter-анимации (AnimatedBuilder) |
| **AI-чат** | YandexGPT API |
| **Уведомления** | flutter_local_notifications |

---

## ✨ Ключевые возможности

### 🔐 Авторизация и регистрация
- Вход по логину и паролю
- 3-шаговая регистрация с подтверждением по SMS
- Валидация пароля (минимум 8 символов, заглавные/строчные буквы, цифры)

### 🖥 Бронирование ПК
- Выбор клуба из списка
- Календарь с выбором даты и времени
- Выбор длительности сессии (30 мин — 5 часов)
- Интерактивная карта ПК с отображением занятости
- Умный расчёт цены на основе зоны (Bootcamp, VIP, GameZone)
- **Session Hub** — быстрый доступ к активным сессиям

### 📊 История бронирований
- Просмотр всех сессий
- Статусы: UPCOMING, ACTIVE, EXPIRED
- Возможность отмены будущих бронирований

### 👤 Профиль
- Отображение баланса
- Пополнение счёта (с обработкой ошибок сервера)
- Настройки темы (светлая/тёмная)
- Переключение языка (русский/английский)

### 🤖 AI-чат поддержки
- Интеграция с YandexGPT
- Контекстные ответы на основе документации клуба
- Быстрые ответы на частые вопросы

### 📰 Новости
- Лента новостей клуба
- Парсинг HTML-контента
- Pull-to-refresh

### 🎬 Видеоплеер
- Встроенный плеер для трансляций
- Поддержка YouTube

## 🎨 Дизайн и UX

- **Тёмная и светлая темы** с автоматическим переключением
- **Кастомные анимации**:
  - 🐻 **BearLogoAnimation** — анимация логотипа с медведем (появление + свечение + пульсация)
  - 🎮 **GamingSpinner** — вращающийся геймерский спиннер для загрузки
  - ✅ **SuccessAnimation** — анимация успеха с конфетти и пульсирующей галочкой
  - ⏳ **LoadingDots** — анимированные точки для микро-загрузок
- **Shimmer-скелетоны** для загрузки списков
- **Адаптивный дизайн** под разные размеры экранов

---

## 🛡 Надёжность и отказоустойчивость

- **Оффлайн-режим**: кеширование данных через Hive
- **Индикатор подключения**: отображение статуса сети
- **Обработка ошибок сервера**: все API-запросы обёрнуты в try-catch
- **Умная проверка ответов**: сервер может возвращать "success" с кодом 0, 3 или 200
- **Локальное состояние**: бронирования сохраняются локально при сбоях сервера
- **Push-уведомления**: напоминания о начале и конце сессии

## 🛠 Технические детали

### API
- **Base URL**: `https://vibe.blackbearsplay.ru`
- **Формат**: JSON (application/json)
- **Аутентификация**: member_name + password → сессионный ключ
- **Подпись запросов**: MD5 (rand_key + key)

### Структура проекта
```
lib/
├── main.dart              # Точка входа
├── core/
│   ├── config.dart        # Конфигурация
│   ├── services/          # Сервисы (кеш, уведомления)
│   └── utils/             # ApiClient
├── models/                # Модели данных
├── providers/             # State management
├── screens/               # Экраны
└── widgets/               # Переиспользуемые виджеты
```

### Зависимости
- `provider` — управление состоянием
- `http` — HTTP-запросы
- `google_fonts` — шрифты (Montserrat)
- `font_awesome_flutter` — иконки
- `curved_navigation_bar` — навигация
- `shared_preferences` + `hive` — локальное хранение
- `connectivity_plus` — статус сети
- `shimmer` — скелетоны загрузки
- `flutter_local_notifications` — уведомления
- `flutter_html` — парсинг HTML
- `webview_flutter` — встроенный браузер

## 🏆 Почему BBplay достоин победы?

1. **Полноценное production-приложение** с реальной серверной интеграцией
2. **Сложная бизнес-логика**: бронирование, расчёт цен, работа с нестабильным API
3. **Отказоустойчивость**: оффлайн-режим, кеширование, обработка ошибок
4. **AI-интеграция**: чат с YandexGPT для поддержки пользователей
5. **Кастомные анимации**: уникальный UI с анимированным логотипом
6. **Push-уведомления**: напоминания о сессиях
7. **Кросс-платформенность**: Android, iOS, Web
8. **Тёмная/светлая тема** и локализация (RU/EN)
